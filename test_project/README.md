# Test Project - Sample Application with Code Issues

This is a simple Python web application with intentional security, performance, and style issues for testing the Code Review Agent.

## Files:
- `app.py` - Flask web application (security issues)
- `database.py` - Database operations (SQL injection, N+1 queries)
- `utils.py` - Helper functions (performance and style issues)
- `config.py` - Configuration (hardcoded secrets)
